## 1. What are we learning?

Today in science we will learn how to plan a short experiment — with a hypothesis, observation, and one variable.

---

## 2. Simple explanation

A scientific experiment tests a question about the world.
Basic steps:
First step — question: what do we want to test?
Second step — hypothesis: what do we think will happen?
Third step — experiment: change only one thing (the variable).
Fourth step — recording: a table with results.
Fifth step — conclusion: what did we learn?
Variable — the only thing you change (for example amount of water).
Everything else stays the same — so the test is fair.
Safe experiments only:
Observation, comparison, measurement with a teacher — no dangerous materials.

---

## 3. Example

Do seeds sprout faster with more water?
Hypothesis:
With more water — faster sprouting.
Experiment:
Two identical cups, same number of seeds, same location.
Cup A — a little water; Cup B — more water (variable).
Write in a table how many seeds sprouted each day.
Conclusion:
According to the data — do the data support the hypothesis?

---

## 4. Let's solve together

Why change only one thing in an experiment?
If you change several things — you do not know what made the difference.
One variable gives a clear result.
So we know what caused the change

---

## 5. Try it yourself

Plan a safe experiment:
Which plant is taller — by a window or in a corner?
What is the hypothesis?
What is the variable?
What stays the same?
Try to solve it on your own.
On the next page we will check the steps and the answer together.

---

## 6. Let's check together

In this experiment the only variable is light location (by a window or in a dark corner).
Everything else is the same — same type of plant, same amount of water, same soil.
Only then can we know for sure that light location is the cause of the height change.
Hypothesis — plant by a window will grow taller; variable — light location.
A fair experiment is one with only one variable, and all other conditions identical.
