## 2. הסבר

בישראל מים יקרים — צריך לחסוך.
קרקע — לגידולים ולבנייה — צריך לשמור.
אנרגיה — לחשמל ולתחבורה — לפעמים מהשמש.
המדינה והקהילה דואגים לניהול — חוקים, תכנון, חיסכון.
גם אנחנו — סוגרים ברז, ממחזרים — חלק מהניהול.
