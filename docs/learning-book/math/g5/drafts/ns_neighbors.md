## 2. הסבר

שכן אחרי = +1
45,600:
- שלפני: 45,599
- אחרי: 45,601
כשיש אפסים: 45,600 − 1 = 45,599 — לא 45,500!

ערך המקום: רק ספרת האחדות משתנה ב-±1 — אבל כל הספרות יכולות "להתגלגל" קדימה או אחורה.
