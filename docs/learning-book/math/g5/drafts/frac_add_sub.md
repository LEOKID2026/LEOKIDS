## 1. What are we learning?

Today we'll learn to add and subtract fractions with a common denominator.
You only work with the numerators — the denominator stays the same.

---

## 2. Simple explanation

Common denominator → add/subtract numerators
3/8 + 2/8 = 5/8
(denominator 8 — add numerators 3+2=5)
3/8 − 2/8 = 1/8

---

## 3. Visual / concrete example

5/6 − 1/6 = 4/6 = 2/3 (you can simplify)

---

## 4. Let's solve together

Work out 3/8 + 2/8 = ?
Common denominator 8 — already there
3 + 2 = 5
5/8

## 6. Common mistake — watch out!

Work out: 4/9 + 3/9 = ?
Common denominator 9 — already there
Add numerators: 4 + 3 = 7
The denominator stays 9
4/9 + 3/9 = 7/9
