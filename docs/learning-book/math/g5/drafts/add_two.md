## 1. What are we learning?

Today we'll learn to add two numbers up to 100,000.
We'll break them into ten-thousands, thousands, hundreds, tens, and ones — and remember to carry when a sum goes past 10.

---

## 2. Simple explanation

Steps for addition:
Break each number apart by place value
Add the ones — if you get 10 or more, carry 1 to the tens
Add the tens (including the carry) — carry to the hundreds
Add the hundreds — carry to the thousands
Add the thousands — carry to the ten-thousands
Add the ten-thousands
24,680 + 5,415:
Ones: 0+5=5; tens: 8+1=9; hundreds: 6+4=10→0+carry 1; thousands: 4+5+1=10→0+carry 1; ten-thousands: 2+0+1=3

## 3. Visual / concrete example

## 4. Let's solve together

Work out 38,450 + 16,275 = ?
Ones: 0 + 5 = 5
Tens: 5 + 7 = 12 → write 2, carry 1
Hundreds: 4 + 2 + 1 = 7
Thousands: 8 + 6 = 14 → write 4, carry 1
Ten-thousands: 3 + 1 + 1 = 5
38,450 + 16,275 = 54,725

## 6. Common mistake — watch out!

Work out: 47,350 + 6,284 = ?
Ones: 0 + 4 = 4
Tens: 5 + 8 = 13 → 3, carry 1
Hundreds: 3 + 2 + 1 = 6
Thousands: 7 + 6 = 13 → 3, carry 1
Ten-thousands: 4 + 0 + 1 = 5
47,350 + 6,284 = 53,634
