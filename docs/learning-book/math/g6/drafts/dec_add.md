## 6. Common mistake — watch out!

Calculate: 3.67 + 2.85
Line up the decimal points.
Hundredths: 7+5=12 → 2, carry 1.
Tenths: 6+8+1=15 → 5, carry 1.
Ones: 3+2+1=6
The answer: 6.52
