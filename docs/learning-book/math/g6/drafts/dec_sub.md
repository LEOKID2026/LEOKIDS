## 6. Common mistake — watch out!

Calculate: 8.40 − 3.75
Hundredths: 0<5 → borrow from tenths. 10−5=5.
Tenths: 3 (after borrowing) < 7 → borrow from ones. 13−7=6.
Ones: 7 (after borrowing) − 3 = 4.
Check: 4.65 + 3.75 = 8.40
The answer: 4.65
