## 1. What are we learning?

Today we will learn to add two numbers up to 200,000.
Work from ones → tens → hundreds → thousands → ten thousands → hundred thousands, with carrying.

---

## 2. Simple explanation

12,450 + 3,875:
- ones: 0+5=5
- tens: 5+7=12 → 2, carry 1
- hundreds: 4+8+1=13 → 3, carry 1
- thousands: 2+3+1=6
- ten thousands: 1+0=1
= 16,325

---

## 3. Visual / concrete example

48,750 + 25,680 = 74,430
Check: 74,430 − 25,680 = 48,750

---

## 4. Let's solve together

Calculate 48,750 + 25,680 = ?
Ones: 0 + 0 = 0
Tens: 5 + 8 = 13 → write 3, carry 1
Hundreds: 7 + 6 + 1 = 14 → write 4, carry 1
Thousands: 8 + 5 + 1 = 14 → write 4, carry 1
Ten thousands: 4 + 2 + 1 = 7
48,750 + 25,680 = 74,430

## 6. Common mistake — watch out!

Calculate: 34,267 + 15,486
Ones: 7+6=13 → 3, carry 1.
Tens: 6+8+1=15 → 5, carry 1.
Hundreds: 2+4+1=7
Thousands: 4+5=9
Ten thousands: 3+1=4
The answer: 49,753
