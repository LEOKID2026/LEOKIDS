## 1. מה לומדים?

היום נלמד לחבר מספרים עשרוניים — עם נקודה (למשל 2.3 + 1.4).
חשוב: ליישר את הנקודות — חלק שלם מתחת לשלם, עשיריות מתחת לעשיריות.

---

## 2. הסבר

2.3 + 1.4
2 + 1 = 3 (שלם)
0.3 + 0.4 = 0.7 (אחרי הנקודה)
2.3 + 1.4 = 3.7
כלל: מחברים כאילו אין נקודה, ואז שמים נקודה באותו מקום.

---

## 3. דוגמה

4.2 + 3.5
שלם: 4 + 3 = 7
עשיריות: 2 + 5 = 7
4.2 + 3.5 = 7.7

---

## 4. בואו נפתור

חשבו: 1.6 + 2.2 = ?
שלם: 1 + 2 = 3
עשיריות: 6 + 2 = 8 → 0.8
1.6 + 2.2 = 3.8

## 5. נסו בעצמכם

חשבו: 3.6 + 2.1 = ?

## 6. בואו נבדוק יחד
