## 1. מה לומדים?

היום נמצא שכנים במספרים גדולים — עד 10,000.
±1 גם כשעוברים אלף — למשל 1,999 → 2,000 → 2,001.

---

## 2. הסבר

שכן שלפני = −1
שכן אחרי = +1
2,450:
- שלפני: 2,449
- אחרי: 2,451
מעבר אלף:
- 1,999 → אחרי = 2,000
- 2,000 → שלפני = 1,999
ערך המקום: כשעוברים אלף, כל הספרות יכולות להשתנות — אבל הקפיצה עדיין 1.

---

## 3. דוגמה

3,628:
3,628 − 1 = 3,627
3,628 + 1 = 3,629
2,999 → 3,000: שכן אחרי ל-2,999 = 3,000 — מעבר אלף, +1 בלבד.
3,626 — 3,627 — [3,628] — 3,629 — 3,630
