## 1. What are we learning?

Today we'll learn sequences with big jumps — not just +1 or +5, but jumps like +50 or +100.
Find the rule and keep going — up to 10,000.

---

## 2. Simple explanation

A sequence = numbers in a row following a rule.
Fixed jump: each step you add (or subtract) the same number.
100, 150, 200, 250, …
- 150 − 100 = 50
- 200 − 150 = 50
- The jump: +50 each time
The next number: 250 + 50 = 300

---

## 3. Visual / concrete example

3,250, 3,300, 3,350, 3,400, ?
3,300 − 3,250 = 50
3,350 − 3,300 = 50
3,400 − 3,350 = 50
Jump: +50
3,400 + 50 = 3,450
Answer: 3,450

---

## 4. Let's solve together

Continue: 2,800, 2,850, 2,900, 2,950, ?
2,850 − 2,800 = 50 The jump is +50.
2,950 + 50 = 3,000
3,000

## 6. Common mistake — watch out!

Continue the sequence: 1,000, 1,100, 1,200, ?
1,100 − 1,000 = 100 The jump is +100.
Check: 1,200 − 1,100 = 100 — same jump.
1,200 + 100 = 1,300
1,300
