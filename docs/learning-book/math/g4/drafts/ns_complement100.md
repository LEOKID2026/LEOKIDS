## 2. הסבר

משלימים ל-100: כמה צריך להוסיף כדי שהסכום יהיה 100?
37 + ? = 100 → חסרים 63 (כי 37 + 63 = 100)
אסטרטגיה — "קודם ל-100" בחיבור:
1.
מוצאים כמה חסר לעשרה עגולה הקרובה (למשל 37 + 3 = 40)
2.
או משלימים ישר ל-100 (למשל 68 + 32 = 100)
3.
ממשיכים עם מה שנשאר
דוגמה: 47 + 58
- 47 + 3 = 50, נשאר 55 מ-58 → 50 + 55 = 105
דרך פשוטה: 37 + 63 = 100 — שמרו את הזוג בזיכרון.

---

## 3. דוגמה

חיבור בראש עם השלמה:
68 + 35 = ?
68 + 32 = 100 (הזוג 68 + 32)
נשאר 3 מ-35 → 100 + 3 = 103
68 + 35 = 103

## 4. בואו נפתור

37 + ? = 100 — מה המספר החסר?
100 − 37 = 63
37 + 63 = 100 — הזוג 37 ו-63.
בדיקה: 30 + 70 = 100, ו-7 + 3 = 10 → 100
המספר החסר הוא 63.
