## 1. What are we learning?

Today we'll learn a strategy for mental addition using making 100.
Know the pair 37 + 63 = 100?
You can use it in big additions.

---

## 2. Simple explanation

Make 100: how much do you need to add so the sum is 100?
37 + ? = 100 → you need 63 (because 37 + 63 = 100)
Strategy — "get to 100 first" in addition:
1.
Find how much is missing to the nearest round ten (for example 37 + 3 = 40)
2.
Or go straight to 100 (for example 68 + 32 = 100)
3.
Keep going with what's left
Example: 47 + 58
- 47 + 3 = 50, 55 left from 58 → 50 + 55 = 105
Easy way: 37 + 63 = 100 — remember that pair!

---

## 3. Visual / concrete example

Mental addition with making 100:
68 + 35 = ?
68 + 32 = 100 (the pair 68 + 32)
3 left from 35 → 100 + 3 = 103
68 + 35 = 103

---

## 4. Let's solve together

37 + ? = 100 — what is the missing number?
100 − 37 = 63
37 + 63 = 100 — the pair 37 and 63.
Check: 30 + 70 = 100, and 7 + 3 = 10 → 100
The missing number is 63.

## 6. Common mistake — watch out!

64 + ? = 100 — what is the missing number?
100 − 64 = 36
Check: 64 + 36 = 100
The missing number is 36
