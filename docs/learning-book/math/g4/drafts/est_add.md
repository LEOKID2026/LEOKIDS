## 2. הסבר

487 + 512 ≈ ?
עיגול כל מספר למספר נוח (למשל למאה הקרובה):
487 ≈ 500
512 ≈ 500
500 + 500 = 1,000
487 + 512 ≈ 1,000
למה? 487 קרוב ל-500, 512 קרוב ל-500 — והסכום בערך 1,000.
הערכה עוזרת לבדוק אם תשובה מדויקת סבירה.

## 3. דוגמה

298 + 403 ≈ ?
298 ≈ 300
403 ≈ 400
300 + 400 = 700
298 + 403 ≈ 700
(בדיוק: 701 — קרוב מאוד!)

## 4. בואו נפתור

356 + 241 ≈ ?
356 ≈ 360 (או 400).
241 ≈ 240 (או 200).
360 + 240 = 600 (או 400 + 200 = 600).
356 + 241 ≈ 600

## 5. נסו בעצמכם

623 + 382 ≈ ?
נסו לפתור בעצמכם.
בעמוד הבא נבדוק יחד את הדרך ואת התשובה.

## 6. בואו נבדוק יחד

623 + 382 ≈ ?
עיגול: 623 ≈ 600.
עיגול: 382 ≈ 400.
600 + 400 = 1,000
623 + 382 ≈ 1,000
