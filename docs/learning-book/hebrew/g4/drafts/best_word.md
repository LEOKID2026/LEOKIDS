## 2. הסבר

שאלו:
מה בדיוק קורה?
חזק או חלש?
חינוכי או סיפורי?
קראו את כל המשפט — ובחרו מילה אחת.
מדויק — לא "בערך".

## 3. דוגמה

קטע:
המדריך במוזיאון נתן הסבר מפורט על הממיות — עם תאריכים ושמות מקומות.
המילה הטובה:
מפורט — לא רק "יפה" או "ארוך".

## 4. בואו נפתור

השלימו.

קראו — מדויק ומתאים.
