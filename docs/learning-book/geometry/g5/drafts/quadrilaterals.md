## 2. Simple explanation

:::geometry-diagram
type: quadrilateral_parts
:::

Parallelogram тЖР rectangle тЖР square (inclusion chain)
Trapezoid тАФ separate: only one pair parallel (not 2 pairs).
Parallelogram тАФ 2 pairs of parallel sides.
Trapezoid тАФ 1 pair (the bases).
Rectangle тАФ 2 pairs + 90┬░.
Square тАФ 2 pairs + equal sides.
