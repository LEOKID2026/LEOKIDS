## 1. מה לומדים?

היום נעמיק למה סכום זוויות במשולש הוא 180° — ונשתמש בזה כדי למצוא זווית חסרה.
רעיון: אם "גוזרים" את שלוש הזוויות ומצמידים אותן — הן יוצרות קו ישר (180°).

---

## 2. הסבר

:::geometry-diagram
type: angle_basic
:::

במשולש ABC:
- זווית A + זווית B + זווית C = 180°
למה? שלוש הזוויות "ממלאות" יחד פתיחה של קו ישר — לכן הסכום תמיד 180°.
אם יודעים A ו-B — מוצאים C:
C = 180° − (A + B)

---

## 3. דוגמה

במשולש ABC: A = 60°, B = 70°
נימוק: 60° + 70° = 130° — "כמה נשאר" עד 180°?
180° − 130° = 50° → C = 50°
בדיקה: 60° + 70° + 50° = 180°  — שלוש הזוויות "סוגרות" קו ישר.

---

## 4. בואו נפתור

במשולש ABC זווית A = 60°, זווית B = 70°. מה זווית C?
שלב 1 (נימוק): שתי הזוויות יחד = 60° + 70° = 130°.
שלב 2 (חסר): 180° − 130° = 50°
שלב 3 (בדיקה): 60° + 70° + 50° = 180°
50°

---

## 5. נסו בעצמכם

במשולש ABC: 45° ו-95° — מה הזווית השלישית?
נסו לפתור בעצמכם.
בעמוד הבא נבדוק יחד את הדרך ואת התשובה.

---

## 6. בואו נבדוק יחד

במשולש ABC: זווית A = 45°, זווית B = 95°— מה זווית C?
חיבור שתי הזוויות הידועות:
45° + 95° = 140°
כמה נשאר מ-180°?
180° − 140° = 40°
בדיקה: 45° + 95° + 40° = 180°
זווית C = 40°

---

## 7. בואו נתרגל!

סכום 180° במשולש הוא כלל עם נימוק — לא רק נוסחה.
בתרגול תמצאו זווית חסרה — חברו, חסרו מ-180°, ובדקו שהסכום 180°!
