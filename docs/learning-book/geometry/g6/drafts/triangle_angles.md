## 1. מה לומדים?

היום נפתור זווית חסרה במשולש בשלבים — עם משוואה ואות x לזווית הלא ידועה.
לא רק "180° פחות חיבור" — אלא: x + זווית 1 + זווית 2 = 180°.

---

## 2. הסבר

:::geometry-diagram
type: angle_basic
:::

במשולש ABC, נסמן את הזווית החסרה ב-x.
משוואה: x + A + B = 180°
שלבים:
1.
מציבים את הזוויות הידועות
2.
מחברים את הידועות
3.
פותרים: x = 180° − (A + B)
4. בודקים: x + A + B = 180°

---

## 3. דוגמה

במשולש: 55°, 65° ו-x (חסרה).
משוואה: x + 55° + 65° = 180°
שלב א: 55° + 65° = 120°
שלב ב: x + 120° = 180°
שלב ג: x = 180° − 120° = 60°
בדיקה: 60° + 55° + 65° = 180°

---

## 4. בואו נפתור

במשולש ABC זווית A = 55°, זווית B = 65°. מצאו זווית C — סמנו x.
x + 55° + 65° = 180°
55° + 65° = 120° → x + 120° = 180°
x = 180° − 120° = 60°
שלב 4 (בדיקה): 60° + 55° + 65° = 180°
C = 60° (x = 60°)

---

## 5. נסו בעצמכם

במשולש: 40° ו-65° — כתבו משוואה x + 40° + 65° = 180° ומצאו x.
נסו לפתור בעצמכם.
בעמוד הבא נבדוק יחד את הדרך ואת התשובה.

---

## 6. בואו נבדוק יחד

במשולש: זוויות 40° ו-65° — מצאו את הזווית השלישית x.
x + 40° + 65° = 180°
40° + 65° = 105° → x + 105° = 180°
x = 180° − 105° = 75°
שלב 4 (בדיקה): 75° + 40° + 65° = 180°
x = 75°

---

## 7. בואו נתרגל!

כאן פותרים זווית חסרה עם משוואה ובדיקה.
בתרגול תמצאו x +… = 180° — חברו ידועות, פתרו x, ובדקו!
