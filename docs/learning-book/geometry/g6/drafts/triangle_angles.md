## 1. What are we learning?

Today we will find a missing angle in a triangle step by step — with an equation and x for the unknown angle.
Not just "180° minus addition" — but: x + angle 1 + angle 2 = 180°.

---

## 2. Simple explanation

:::geometry-diagram
type: angle_basic
:::

In triangle ABC, we label the missing angle x.
Equation: x + A + B = 180°
Steps:
1.
Plug in the known angles
2.
Add the known ones
3.
Solve: x = 180° − (A + B)
4. Check: x + A + B = 180°

---

## 3. Example

In a triangle: 55°, 65°, and x (missing).
Equation: x + 55° + 65° = 180°
Step a: 55° + 65° = 120°
Step b: x + 120° = 180°
Step c: x = 180° − 120° = 60°
Check: 60° + 55° + 65° = 180°

---

## 4. Let's solve together

In triangle ABC angle A = 55°, angle B = 65°. Find angle C — label it x.
x + 55° + 65° = 180°
55° + 65° = 120° → x + 120° = 180°
x = 180° − 120° = 60°
Step 4 (check): 60° + 55° + 65° = 180°
C = 60° (x = 60°)

---

## 5. Try it yourself

In a triangle: 40° and 65° — write the equation x + 40° + 65° = 180° and find x.
Try to solve by yourself.
On the next page we'll check the steps and the answer together.

---

## 6. Let's check together

In a triangle: angles 40° and 65° — find the third angle x.
x + 40° + 65° = 180°
40° + 65° = 105° → x + 105° = 180°
x = 180° − 105° = 75°
Step 4 (check): 75° + 40° + 65° = 180°
x = 75°
