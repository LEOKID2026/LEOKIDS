# Parent report — Editorial & product sign-off (Hebrew overhaul)

מסמך זה משלים את שער **gate-6** בתוכנית שדרוג העברית: אישור תוכן/מוצר/QA לפני שחרור.

## תנאי קדם (אוטומטי)

- [ ] `npm run test:parent-report-hebrew-language` — ירוק (סריקת אסורים ASCII + **readability** למונחים כמו «מאסטרי» / «טקסונומיה» + דוגמאות gloss + `buildWhyThisRecommendationHe`).
- [ ] `npm run test:parent-report-phase6` — ירוק (חוזים + SSR לדפי דוח).
- [ ] אין דליפות מזהים טכניים במסך הורה (ראו `PARENT_REPORT_HEBREW_STYLE_GUIDE.md` §3).

## מעבר עריכה ידני (מומלץ סדר)

1. **דוח קצר** (`/learning/parent-report`) — כותרות, סיכום, כרטיסי מקצוע, פס הסבר נושא, מקור אבחון, הדפסה/PDF.
2. **דוח מפורט** — סיכום מנהלים, חוצה־מקצועי, תוכנית בית, יעדים, פרופילי מקצוע, מכתבים.
3. **אחידות** — אותה מילה לאותו מושג (חיזוק / תרגול / העברה) בכל הערוצים.
4. **נתונים דלים** — וידוא שניסוחים לא נשמעים כ“תקלה” אלא כ“עדיין מוקדם”.
