# מרכז עזרה (Help Center)

נתיבים: `/help`, `/help/parents`, `/help/students`, `/help/parent-report`, `/help/subjects`.

## פקודות

| פקודה | תיאור |
|--------|--------|
| `npm run help:build-manifest` | בונה `screenshots-manifest.json` |
| `npm run help:placeholders` | ממלא PNG מינימלי ב־qa-evidence (אם חסר) |
| `npm run help:capture` | לכידת מסכים (Playwright, דמו ADMIN) |
| `npm run help:publish-screenshots` | מעתיק ממניפסט ל־`public/` |
| `npm run help:verify` | בודק שכל הקבצים קיימים |
| `npm run help:screenshots` | manifest → placeholders → publish → verify |
